# Wedding photo site

Site that lets guests upload photos (Cloudinary) and view them in a simple gallery, hosted on Netlify.

## tl;dr (dev)

```bash
git clone https://github.com/Meurlin-Wedding-Photo-Site/photoSiteRepo.git
cd photoSiteRepo
cp .env.example .env           # on windows: copy .env.example .env
npm i
npx netlify-cli dev            # or: netlify dev if you installed it globally
# open http://localhost:8888
